# Knightfall Chess Code of Conduct

## Our Commitment

Knightfall Chess is committed to creating a welcoming, respectful, and inclusive environment for everyone. Whether you're reporting bugs, suggesting new features, submitting pull requests, or simply exploring the project, we ask that all members of the community treat each other with kindness and professionalism.

Our goal is to build not only a modern chess platform but also a positive open-source community where everyone feels comfortable contributing and learning.

---

## Expected Behavior

We encourage all community members to:

- Be respectful and courteous to others.
- Welcome constructive feedback and different perspectives.
- Communicate professionally and patiently.
- Help new contributors learn and improve.
- Focus discussions on improving the project.
- Respect everyone's time and effort.

---

## Unacceptable Behavior

The following behaviors are not acceptable within the Knightfall Chess community:

- Harassment, bullying, or personal attacks.
- Discriminatory language or hateful conduct.
- Offensive comments related to race, ethnicity, nationality, religion, gender, age, disability, or sexual orientation.
- Threats or intimidation of any kind.
- Sharing another person's private information without permission.
- Spam, trolling, or intentionally disruptive behavior.
- Any behavior that creates an unsafe or unwelcoming environment.

---

## Scope

This Code of Conduct applies to all official Knightfall Chess community spaces, including:

- GitHub Issues
- GitHub Discussions
- Pull Requests
- Project documentation
- Future community platforms associated with Knightfall Chess

---

## Enforcement

Project maintainers reserve the right to edit, remove, or reject comments, issues, pull requests, or other contributions that violate this Code of Conduct.

Depending on the severity of the violation, actions may include:

- A private warning
- Removal of inappropriate content
- Temporary restriction from participating
- Permanent removal from the project's community
