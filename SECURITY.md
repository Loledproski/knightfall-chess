# Security Policy

## Supported Versions

Knightfall Chess is currently under active development.

| Version | Supported |
|----------|-----------|
| Latest (main) | ✅ |
| Older versions | ❌ |

Only the latest version available on the `main` branch receives security fixes and updates.

---

## Reporting a Vulnerability

If you discover a security vulnerability in Knightfall Chess, please do **not** open a public GitHub issue.

Instead, report it privately by contacting:

- GitHub: https://github.com/Loledproski
- Email: harshkumarmishra10@gmail.com

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- Screenshots or videos (if applicable)
- Suggested fixes (optional)

I will investigate all valid reports and work to release a fix as quickly as possible.

Thank you for helping keep Knightfall Chess secure.
